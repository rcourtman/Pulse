# Localized Pulse docs

This directory contains first-wave localized getting-started docs for public
Pulse users.

- [Deutsch](de/README.md)
- [Español](es/README.md)

The canonical deep documentation remains the English docs tree. Localized docs
must preserve commands, image names, config keys, activation-key formats,
paths, API fields, product names, and resource names exactly unless the owning
English source changes.
